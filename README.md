# Fylo Dark Theme Landing Page

A modern, dark-themed landing page built with a focus on responsive design and semantic structure.

[View Project Live](https://gokhansagli.github.io/fylo-dark-theme-landing-page/)

## 🛠️ Tech Stack

- **HTML5:** Semantic markup for SEO and accessibility.
- **CSS3:** Flexbox, CSS Variables, and responsive layouts.
- **jQuery:** Client-side form validation.

## ✨ Key Features

- **Responsive:** Fully compatible with Mobile, Tablet, and Desktop.
- **Validation:** Dynamic email validation for the subscription form.
- **Interactive:** Polished hover effects and interactive elements.
